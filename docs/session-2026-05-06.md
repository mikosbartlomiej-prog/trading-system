**Architektura:**
```
GitHub Actions (co 30 min)
  → defense-monitor/monitor.py
    → scrape defense.gov/News/Contracts/
    → RSS: Defense One, Breaking Defense, Reuters, AP
    → NewsAPI (jeśli NEWSAPI_KEY)
    → scoring: LONG keywords (30+), SHORT keywords (15+)
    → company→ticker mapping
    → jeśli score >= 2 → wyślij alert do Cloudflare
  → Cloudflare Worker: defense-proxy
    → Claude Routine: Defense Market Handler
      → Alpaca MCP → złóż zlecenie
```

**Tickery:**

- Mid-cap (LONG+SHORT): KTOS, PLTR, AXON, LDOS, SAIC, CACI — size $1,500 long / $1,000 short
- ETFs (tylko LONG): ITA, XAR, DFEN — size $2,000
- European ADRs (tylko LONG): BAESY, EADSY — size $1,000

**SL/TP:** -3% / +6% (R:R = 2.0)

**Do zrobienia (ręcznie):**
1. Opcjonalnie: założyć konto na newsapi.org → dodać `NEWSAPI_KEY` do GitHub Secrets
2. Stworzyć Cloudflare Worker `defense-proxy` (kod w SETUP.md)
3. Stworzyć Claude Routine `Defense Market Handler` (prompt w SETUP.md)
4. Dodać secret `CLOUDFLARE_DEFENSE_WORKER_URL` do GitHub
5. `git add defense-monitor/ strategies/defense-market.md .github/workflows/defense-monitor.yml && git commit -m "Add defense market monitor" && git push`
