## Lessons / Obserwacje
- Lokalne klucze API (APCA_API_KEY_ID w środowisku runner) są niezsynchronizowane z aktywnym kluczem Alpaca. GitHub Actions używają innych kluczy (ALPACA_API_KEY secret). Routine nie może wykonać exitów autonomicznie dopóki problem nie będzie naprawiony.
- SPY 5 DTE puts podwójny problem: theta decay + straty. Te powinny być zamknięte nawet bez emergency threshold.
- Recurring pattern: exit-monitor routine odpala poprawnie, payload jest poprawny, ale execution blokuje API auth failure. Systemowy fix potrzebny ASAP.
